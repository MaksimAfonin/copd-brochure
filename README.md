# Роль фельдшера в первичной профилактике ХОБЛ

Минимальный одностраничный сайт для публикации через GitHub Pages.

## Публикация

1. Создать новый репозиторий на GitHub.
2. Загрузить в него файлы `index.html` и `style.css` из этой папки.
3. Открыть в репозитории `Settings` → `Pages`.
4. В `Build and deployment` выбрать:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main`
   - папка: `/root`
5. Нажать `Save`.

Через минуту сайт появится по ссылке, которую GitHub покажет в разделе `Pages`.
